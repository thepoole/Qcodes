qcodes.logger

The qcodes.logger module provides functionality to enable logging of errors and debug information from QCoDeS using the default python logging module.

qcodes.logger.instrument_logger

This module defines a logging.LoggerAdapter and logging.Filter.

qcodes.logger.log_analysis

This module defines a number of functions to make it easier to work with log messages from QCoDeS.

qcodes.logger.logger

This module defines functions to setup the logging of QCoDeS.

The qcodes.logger module provides functionality to enable logging of errors and debug information from QCoDeS using the default python logging module. It also logs command history logging when using IPython/Jupyter.

The module also provides functionality to filter log messages by instrument and functions to extract log messages to pandas.DataFrame s
